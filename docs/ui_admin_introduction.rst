The Admin Interface
----------------------

Introduction
~~~~~~~~~~~~~~~~~~~

The **Operator Admin**, **Regulator User**, **Regulator Admin**, and **Platform Admin** roles have access to the **Administration Console**. 
Each role provides a different permission level within the interface. This manual describes in detail the functionalities available to each of these roles.

  **The Incident User and Operator User roles do not have access to the Administration Console.**

The following section explains how the Operator Admin, Regulator User, Regulator Admin, and Platform Admin can use the Administration Console.
